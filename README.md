# DailyNews


----------


小简报 (知乎日报Android客户端)
## API
知乎日报API来源于：[知乎日报API分析][1]
## 框架
 - 项目框架：[Ted Mosby (MVP模式)][2]
 - 网络框架：[Retrofit2.0][3]
 - 图片框架：[Glide][4]
 - 异步框架：[RxJava][5]  
 - 数据库框架：[Realm][6]
